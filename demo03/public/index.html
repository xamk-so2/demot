Demo 3: REST API

Tämä on demo REST API -palvelinsovelluksesta, jonka tarkoituksena on keskustella toisen sovelluksen kanssa json-muodossa. REST API on rajapinta, joka mahdollistaa palvelimella sijaitsevien tietojen lukemisen, lisäämisen, muokkaamisen ja poiston asiakassovelluksella.

Testaa demon toiminnallisuutta Postman-sovelluksella. Luo Postman-sovelluksella GET-, POST-, PUT- ja DELETE-pyyntöjä url-osoitteeseen http://localhost:3003/api/ostokset testataksesi REST API:n toimintaa.
