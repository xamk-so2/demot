Demo 2: Web Service -perusteita

Tämä on demo Web Service -tyylisestä palvelinsovelluksesta, jonka tarkoituksena on keskustella toisen sovelluksen kanssa json-muodossa. Se siis lähettää eri muotoista json-dataa riippuen pyynnön reitistä (urlista)

Testaa demon toiminnallisuutta Postman-sovelluksella. Luo Postman-sovelluksella GET-pyyntö url-osoitteeseen http://localhost:3002/kayttajatiedot, niin saat kayttajalistauksen json-muodossa. Yksittäisen käyttäjän yhteystiedot saat reitillä http://localhost:3002/yhteystiedot/id, jossa "id" on halutun käyttäjän id, esim. 7.
